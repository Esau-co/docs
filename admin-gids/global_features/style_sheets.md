# Stijlbladen

Zoals vermeld in sectie 4.1.14 Stylesheets op pagina 36, kunnen de Chamilo-stijlen worden bijgewerkt door een eenvoudige kopie van een van de bestaande stijlen in _chamilo/main/css/_. In deze directory vindt u een reeks stijlmappen, bijv. _chamilo\_green_, die naar wens kan worden gekopieerd, hernoemd en gewijzigd.
