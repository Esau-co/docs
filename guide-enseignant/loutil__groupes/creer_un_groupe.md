# Créer un groupe

Depuis la page des groupes :

* cliquer sur le lien « Créer nouveau\(x\) groupe\(s\) »,
* donner le nombre de groupes à créer \(« Confirmer la création de groupe\(s\) »\),
* nommer le groupe, sa catégorie et le nombre de places,
* « Créer groupe\(s\) ».
